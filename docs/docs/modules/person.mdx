# Person

## Gender

`dunna.gender()` returns a random gender male or female

```js
dunna.gender(); // male
dunna.gender(); // female
```

## First name

`dunna.firstName()` returns a random male or female firstName

```js
dunna.firstName(); // Peter
dunna.firstName(); // Hector
dunna.firstName(); // Kathie

// Parameters
dunna.firstName({ gender: 'male' });
```

<Generator
  data={[
    {
      name: 'gender',
      description: 'Specify the gender',
      type: 'enum',
      initialValue: 'any',
      values: ['any', 'male', 'female'],
    },
  ]}
  onClick={(dunna, state, theme) => {
    const text = dunna.firstName({ gender: state.gender });
    return { text, styles: {} };
  }}
/>

## Full Name

`dunna.fullName()` returns a random male or female full name (firstName + lastName)

```js
dunna.fullName(); // Hector Smith
dunna.fullName(); // TODO
dunna.fullName(); // TODO

// Parameters
dunna.fullName({ gender: 'female' }); // Return a random female full name
```

## Email

`dunna.email()` returns a random male or female email address

```js
dunna.email(); // hector_smith@example.com
dunna.email(); // peter_smith@example.com

// Parameters
dunna.email({ gender: 'male' }); // Return a male email
```

<Generator
  data={[
    {
      name: 'gender',
      description: 'Specify the gender',
      type: 'enum',
      initialValue: 'any',
      values: ['any', 'male', 'female'],
    },
    {
      name: 'domain',
      description: 'Specify domain',
      type: 'text',
      initialValue: 'example.com',
    },
  ]}
  onClick={(dunna, state, theme) => {
    const text = dunna.email({ gender: state.gender, domain: state.domain });
    const styles = { width: '300px' };
    return { text, styles };
  }}
/>

```js

// Parameters
```
