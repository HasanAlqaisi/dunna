# Finance

## Currency Name

`dunna.currencyName()` returns a random currency name

```js
dunna.currencyName();
```

<Generator
  onClick={(dunna, state, theme) => {
    const currencyName = dunna.currencyName();
    return { text: currencyName, styles: {} };
  }}
/>

## Currency Code

`dunna.currencyCode()` returns a random currency code

```js
dunna.currencyCode();
```

<Generator
  onClick={(dunna, state, theme) => {
    const currencyCode = dunna.currencyCode();
    return { text: currencyCode, styles: {} };
  }}
/>

## Currency

`dunna.currency()` returns a random currency object (name and code)

```js
dunna.currency();
```
