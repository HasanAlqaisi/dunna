# Location

## Country Name

`dunna.countryName()` returns a random country name

```js
dunna.countryName();
```

<Generator
  onClick={(dunna, state, theme) => {
    const countryName = dunna.countryName();
    return { text: countryName, styles: {} };
  }}
/>
